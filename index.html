OK, that's nice, what is it really?

D3.js is an api wrapper around SVG elements which makes it easy to manipulate multiple data points.

What is SVG?

From Wikipedia:

Scalable Vector Graphics (SVG) is an XML-based vector image format for two-dimensional graphics with support for interactivity and animation. The SVG specification is an open standard developed by the World Wide Web Consortium (W3C) since 1999.

SVG images and their behaviors are defined in XML text files. This means that they can be searched, indexed, scripted, and compressed. As XML files, SVG images can be created and edited with any text editor, but are more often created with drawing software.

Example Transformations:

  • translate - moves the element by x, y coordinates.

Note You can style SVG elements like HTML, and even add classes, but they use different values. Example styling:

  • fill - The color of an element.
  • stroke - Then outline of an element.
  • stroke-width - Then size of the outline.

Quick D3.js primer
  • select/selectAll: Selects HTML elements. The main difference between this and other libraries is that D3's select can also select elements which do not yet exist.
  • data.enter: Binds data to a selected element.
